Algoritmos de Alinhamento¶

Algoritmos de alinhamento de sequências biológicas.

Este módulo implementa algoritmos de alinhamento global (Needleman–Wunsch), alinhamento local (Smith–Waterman) e alinhamento múltiplo progressivo.

bioinf.alinhamento.alinhamento_progressivo(seqs, match=1, mismatch=-1, gap=-1)[source]¶

Alinhamento múltiplo progressivo.

O alinhamento múltiplo progressivo é construído de forma incremental. Inicialmente, as duas primeiras sequências são alinhadas utilizando o algoritmo de Needleman–Wunsch. Em seguida, é calculada uma sequência de consenso, que é alinhada sucessivamente com cada nova sequência. Sempre que surgem gaps no consenso, estes são propagados a todas as sequências já alinhadas, garantindo a consistência do alinhamento.

bioinf.alinhamento.dotplot(seq1, seq2)[source]¶

Dotplot para comparação de sequências.

Este método compara duas sequências carácter a carácter, construindo uma matriz binária onde é assinalada a coincidência entre caracteres nas mesmas posições relativas. O dotplot permite identificar regiões de similaridade e padrões conservados entre sequências.

bioinf.alinhamento.needleman_wunsch_dna(seq1, seq2, match=1, mismatch=-1, gap=-1)[source]¶

Algoritmo de Needleman–Wunsch para alinhamento global.

Este algoritmo realiza o alinhamento global de duas sequências utilizando programação dinâmica. Inicialmente é construída e preenchida uma matriz de scores com base em valores de match, mismatch e gap. De seguida, é efetuado um traceback que permite reconstruir o alinhamento ótimo, podendo introduzir gaps quando necessário.

bioinf.alinhamento.score_blosum(matriz, aa1, aa2)[source]¶

Matriz de substituição e cálculo de score entre sequências.

As matrizes de substituição permitem atribuir um score à substituição de um aminoácido por outro, refletindo a sua probabilidade evolutiva. Neste trabalho é utilizada a matriz BLOSUM62. Com base nesta matriz, é possível calcular o score total entre duas sequências de igual comprimento através da soma dos scores posição a posição.

bioinf.alinhamento.smith_waterman_dna(seq1, seq2, match=1, mismatch=-1, gap=-1)[source]¶

Algoritmo de Smith–Waterman para alinhamento local.

O algoritmo de Smith–Waterman permite identificar as regiões de maior similaridade entre duas sequências. Tal como o Needleman–Wunsch, utiliza programação dinâmica, no entanto, os scores negativos são substituídos por zero, ficando o alinhamento restrito às regiões localmente mais semelhantes.
